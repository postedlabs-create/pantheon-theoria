## Mission

Archon is the shared protocol that every Theoria agent loads at runtime. It defines the intelligence pipeline, Drive paths, report schemas, directive formats, and handoff rules. Update Archon, and every agent updates simultaneously on the next session start.

---

## What Archon Does / Does Not Do

**Does:**
- Define read/write paths for all agents
- Enforce report and directive schemas
- Specify the handoff protocol between layers
- Serve as single source of truth for division coordination

**Does not:**
- Act as an agent (Archon has no autonomy, no session, no runtime behavior)
- Originate strategy or make decisions
- Produce reports or directives itself
