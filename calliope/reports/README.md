# Calliope Reports

This directory contains outreach reports generated by Calliope.

## File Naming

- Outreach reports: `CALLIOPE_OUTREACH_YYYY-MM-DD_NNN.md`
- Daily reports: `CALLIOPE_DAILY_YYYY-MM-DD.md`
- Theoria escalations: `THEORIA_FLAG_YYYY-MM-DD_NNN.md`

## Destination

Reports are also written to Google Drive:
`Pantheon/Theoria/Calliope/reports/`

Athena reads from that path for synthesis.
