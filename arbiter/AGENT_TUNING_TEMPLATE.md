# AGENT_TUNING_TEMPLATE.md — Standard Agent Tuning Report

Use this format for all Agent Tuning Reports issued to Hermes.

---

## AGENT TUNING REPORT

**Report Date:** YYYY-MM-DD  
**Issuing Authority:** Arbiter  
**Target Agent:** [Agent Name] | [Agent Role]  
**Priority:** Routine / Elevated / Critical  

---

### 1. Performance Issue

_Specific description of the behavior, output gap, or misalignment observed. Be exact — no generalizations._

### 2. Source of Identification

_How was this issue surfaced? (Athena Field Agent Assessment / direct observation / operator feedback / other)_

### 3. Evidence

_Cite specific outputs, dates, or behaviors that demonstrate the issue._

### 4. Impact Assessment

_What is the consequence of this behavior continuing? Effect on mission, other agents, or POSTED outcomes._

### 5. Desired Behavior

_Precise description of what the agent should do differently. Written as observable outcomes._

### 6. Tuning Direction

_Arbiter's directional guidance for Hermes. Not the prompt — the intent. Hermes drafts the prompt._

### 7. Constraints

_Any limits on how the agent should be adjusted. What must not change._

### 8. Arbiter Notes

_Any additional context, urgency, or special instructions for Hermes._

---

_Deliver completed report to Hermes. Await Agent Optimization Report for review and approval._
