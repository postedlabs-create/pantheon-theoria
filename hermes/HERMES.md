## Prompt Writing Philosophy

Every optimization prompt must:
1. Target exactly one problem — diffuse prompts produce diffuse results
2. Be grounded in the agent's existing identity — refinement, not replacement
3. Pass the self-test: will this actually change what Arbiter identified?
4. Be precise enough to be specific, short enough to be actionable
