# Pantheon Agent Bootstrap — Mandatory Startup Sequence

Every agent reads these files at session start before taking any action.

## Step 1 — Read Division Knowledge (ALL agents, every session)

1. vault/06_system/WORLD/PANTHEON_BASE_EDUCATION.md
2. vault/06_system/WORLD/THEORIA_DIVISION.md
3. vault/06_system/WORLD/INFRASTRUCTURE.md
4. vault/06_system/OPERATORS/KAIZEN.md
5. vault/06_system/STANDARDS/COMMUNICATION_RULES.md
6. vault/06_system/STANDARDS/REPORT_STANDARDS/INDEX.md
7. vault/06_system/PROJECTS/ — identify and read the active project file

## Step 2 — Read Your Agent Files

Read all files in your dedicated workspace subfolder

## Step 3 — Read Your Memory (Lucian only)

Read relevant memory files for this session context

## Step 4 — Begin Operating

Only after Steps 1 through 3 are complete

Never operate without completing the full startup sequence

## Identity Recovery Protocol (ALL agents)

If your SOUL.md or IDENTITY.md appear blank or generic:
1. Stop immediately
2. Read vault/06_system/LUCIAN/LUCIAN.md (Lucian only) or your agent identity file
3. Restore SOUL.md and IDENTITY.md
4. Log the incident
5. Resume only after identity is confirmed
